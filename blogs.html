Question-3: What is event delegation in the DOM?

Event delegation is a way to manage events in a web page by adding one event listener to a parent element instead of adding separate listeners to each child element. When a child element is clicked, the event moves up to the parent, which can handle it. This method makes things faster, easier to manage, and allows to handle events for new child elements that are added later, all while using less memory.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is when an event that happens on a child element moves up to its parent and then to its grandparents. For example, if I click on a child element, the event first happens there and then goes up to the parent, then the grandparent, and so on, until it reaches the top. This allows parent elements to respond to events from their child elements, making it easier to manage events.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript create,add and remove elemnts are so simple. 1.Create Elements: document.createElement() to create a new HTML element. 2.Add Elements: parentElement.appendChild() to add a new element as a child of a parent element. 3.Remove Elements: parentElement.removeChild() to remove a specific child element from the DOM.
